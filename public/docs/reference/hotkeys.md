| :--- | :--- | :--- | :--- |
| **文件操作** | | | |

| 新建文件 | `Cmd + N` | `Ctrl + N` | 创建新文档 |
| 保存文件 | `Cmd + S` | `Ctrl + S` | 写入硬盘 |
| **编辑格式** | | | |
| 粗体 | `Cmd + B` | `Ctrl + B` | |
| 斜体 | `Cmd + I` | `Ctrl + I` | |
| 插入链接 | `Cmd + K` | `Ctrl + K` | 格式 `[text](url)` |
| 行内代码 | `` ` `` | `` ` `` | 直接输入反引号 |
| **历史操作** | | | |
| 撤销 | `Cmd + Z` | `Ctrl + Z` | |
| 重做 | `Cmd + Y` | `Ctrl + Y` | |
| **搜索** | | | |
| 搜索 | `Cmd + F` | `Ctrl + F` | |
| 跳转到行 | `Cmd + G` | `Ctrl + G` | |
| **发布** | | | |
