# 快捷键列表

## 编辑格式

| 功能 | Mac | Windows / Linux |
| :--- | :--- | :--- |
| 粗体 | `Cmd + B` | `Ctrl + B` |
| 斜体 | `Cmd + I` | `Ctrl + I` |
| 撤销 | `Cmd + Z` | `Ctrl + Z` |
| 重做 | `Cmd + Shift + Z` | `Ctrl + Y` |

## 文件操作

| 功能 | Mac | Windows / Linux | 说明 |
| :--- | :--- | :--- | :--- |
| 保存 | `Cmd + S` | `Ctrl + S` | 手动保存，会显示提示 |

> **提示**：编辑器会每 3 秒自动保存，无需频繁手动保存。
