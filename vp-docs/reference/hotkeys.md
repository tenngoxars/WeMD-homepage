# 快捷键列表

## 常用操作

| 功能 | Mac | Windows / Linux | 说明 |
| :--- | :--- | :--- | :--- |
| **保存** | `Cmd + S` | `Ctrl + S` | 手动保存（编辑器会自动保存，非必须） |
| **搜索** | `Cmd + F` | `Ctrl + F` | 呼出搜索面板 |
| **替换** | `Cmd + Opt + F` | `Ctrl + Shift + F` | 呼出搜索面板并展开替换 |
| **撤销** | `Cmd + Z` | `Ctrl + Z` | |
| **重做** | `Cmd + Shift + Z` | `Ctrl + Y` | |

## 插入与格式

| 功能 | Mac | Windows / Linux | 说明 |
| :--- | :--- | :--- | :--- |
| **粗体** | `Cmd + B` | `Ctrl + B` | 选中文字或光标处插入 |
| **斜体** | `Cmd + I` | `Ctrl + I` | |
| **下划线** | `Cmd + U` | `Ctrl + U` | |
| **删除线** | `Cmd + Shift + X` | `Ctrl + Shift + X` | |
| **行内代码** | `Cmd + Shift + K` | `Ctrl + Shift + K` | 插入 \`Code\` |
| **行内公式** | `Cmd + Shift + M` | `Ctrl + Shift + M` | 插入 $LaTeX$ |
| **插入链接** | `Cmd + K` | `Ctrl + K` | |
| **插入图片** | `Cmd + Ctrl + I` | `Ctrl + Alt + I` | |

## 段落与排版

| 功能 | Mac | Windows / Linux | 说明 |
| :--- | :--- | :--- | :--- |
| **一至四级标题** | `Cmd + 1...4` | `Ctrl + 1...4` | 切换当前行的一至四级标题 |
| **无序列表** | `Cmd + Opt + L` | `Ctrl + Alt + L` | 切换 `- List` |
| **有序列表** | `Cmd + Opt + O` | `Ctrl + Alt + O` | 切换 `1. List` |
| **引用块** | `Cmd + Shift + .` | `Ctrl + Shift + .` | 切换 `> Quote` |
| **代码块** | `Cmd + Opt + C` | `Ctrl + Alt + C` | 插入 \`\`\` block |
| **公式块** | `Cmd + Opt + M` | `Ctrl + Alt + M` | 插入 $$ block $$ |
| **插入表格** | `Cmd + Opt + T` | `Ctrl + Alt + T` | 插入 2x2 表格 |
| **插入分割线** | `Cmd + Opt + -` | `Ctrl + Alt + -` | 插入 `---` |
| **缩进** | `Tab` | `Tab` | 列表或引用缩进 |

> **提示**：除了快捷键，WeMD 还支持标准的 Markdown 语法输入，例如输入 `# ` 自动变标题，`- ` 自动变列表等。
