# 快速上手

本文档将引导你在 5 分钟内完成从「打开 WeMD」到「发布一篇排版优雅的公众号文章」的全过程。

## 目标
- 使用 WeMD 编写一篇 Markdown 文章
- 完成排版并复制到微信公众号

---

## 步骤 1：开始写作

### 🌐 Web 网页版
1. 打开 [WeMD 网页版](https://edit.wemd.app)。
2. 页面会直接显示编辑器，默认使用**浏览器存储模式**。

#### 关于存储模式
Web 版提供两种存储模式，点击顶部导航栏的 **「存储模式」** 按钮可切换：

| 模式 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| **浏览器存储** | 文章保存在浏览器内部 (IndexedDB)，关闭网页后文章依然存在 | 临时写作、不想授权文件夹 |
| **本地存储** | 直接读写指定文件夹中的 `.md` 文件，体验接近桌面版 | 长期使用、需要与其他应用共享文件 |

> ⚠️ **注意**：
> - **浏览器存储**：只有在「清除浏览数据」中勾选「Cookie 及其他网站数据」并清除时才会删除文章。
> - **本地存储**：需要浏览器支持 File System Access API。

**本地存储模式浏览器兼容性**：

| 浏览器 | 支持情况 |
| :--- | :--- |
| Chrome / Edge | ✅ 原生支持 |
| Safari | ❌ 不支持 |
| Firefox | ⚠️ 需手动开启（见下方） |

<details>
<summary>🦊 Firefox 开启方法</summary>

1. 地址栏输入 `about:config`，回车。
2. 点击「接受风险并继续」。
3. 搜索 `dom.fs.enabled`，双击将其设为 `true`。
4. 重启 Firefox。

*注：此为实验性功能，可能不稳定。*
</details>

<details>
<summary>🌐 第三方 Chromium 浏览器（如 Arc、Brave 等）</summary>

部分基于 Chromium 的浏览器可能默认禁用此 API：

1. 地址栏输入 `chrome://flags`，回车。
2. 搜索 `File System Access API`。
3. 将其设为 `Enabled`。
4. 点击「Relaunch」重启浏览器。

</details>

### 🖥️ Desktop 桌面版
1. 打开 WeMD 桌面应用。
2. 首次打开会看到欢迎页面，点击 **「选择工作区文件夹」** 按钮。
3. 在系统文件选择器中选择文件夹，无需额外授权。

### 🐳 Docker 部署

如需在自己的服务器上部署 WeMD：

```bash
# 克隆项目
git clone https://github.com/tenngoxars/WeMD.git
cd WeMD

# 一键启动
docker-compose up -d
```

访问 `http://你的服务器IP:8080` 即可使用。

---

## 步骤 2：创建文档

1. 在左侧文件列表/历史记录顶部，点击 **`+`** 按钮创建新文档。
2. 在编辑器中输入以下 Markdown 内容：

```markdown
# 这是一个标题
这里是一段正文。WeMD 让排版变得**如此简单**。
```

---



## 步骤 3：排版与预览

1. 看向右侧的 **预览区**。你会发现刚才输入的文字已经自动变成了公众号的样式。
2. 点击顶部导航栏的 **「主题管理」** 按钮。
3. 在弹出的面板中，选择 **「内置主题」** 列表中的任意主题，如「学术论文」或「赛博朋克」。
4. 预览区会实时更新样式。

---

## 步骤 4：发布

1. 点击顶部导航栏的 **「复制到公众号」** 按钮。
2. 打开微信公众号后台编辑器。
3. 按下 `Cmd+V`（Windows 为 `Ctrl+V`）粘贴。

**🎉 恭喜！** 你已经成功使用 WeMD 发布了第一篇排版优美的文章。

---

## Web 与 Desktop 功能差异

| 功能 | Web 网页版 | Desktop 桌面版 |
| :--- | :--- | :--- |
| 默认存储 | 浏览器内 (IndexedDB) | 本地文件夹 |
| 存储模式切换 | ✅ 顶部「存储模式」按钮可切换 | ❌ 无，固定使用本地文件夹 |
| 文件变更监听 | ❌ 不支持 | ✅ 外部修改自动刷新 |

---

## 💡 进阶技巧

- **深色模式预览**：点击顶部导航栏的 🌙 图标，可预览文章在微信深色模式下的效果。
- **高级搜索**：按下 `Cmd+F` (Windows 为 `Ctrl+F`) 可呼出搜索面板，支持正则匹配与批量替换。
- **外链转脚注**：在工具栏中开启「外链转脚注」开关，文章中的链接将自动转换为文末脚注形式。
- **水平滑动图片**：使用 `<![Alt](url),![Alt](url)>` 语法可生成横向滑动的图片组，适合多图展示。
