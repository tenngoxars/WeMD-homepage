# WeMD 文档

欢迎来到 WeMD 文档中心。

WeMD 是一款专为微信公众号设计的 Markdown 编辑器，支持一键复制到公众号、多种精美主题、深色模式预览等功能。

## 核心功能

| 功能 | 说明 |
|------|------|
| **一键复制** | 完美兼容微信公众号 |
| **多主题** | 12+ 精美主题可选 |
| **深色模式预览** | 预览微信深色模式效果 |
| **本地优先** | 数据存储在本地，隐私安全 |

## 技术亮点

WeMD 内置了 [微信深色模式预览算法](/reference/dark-mode-algorithm)，可在发布前预览文章在深色模式下的效果，还原度约 90%。
